üéØ Queues - First In, First Out (FIFO)

What is a Queue?

A queue is like a line at a store - the first person in line is the first person served! This is called FIFO (First In, First Out).

Real-world examples:

  • üñ®Ô∏è Print queue (first document sent gets printed first)
  • üìû Customer service call queue
  • üé¨ Breadth-First Search in graphs/trees
  • üì® Message queues in distributed systems
  • üéÆ Game matchmaking queues
Key Operations:
  • Enqueue: Add element to rear - O(1)
  • Dequeue: Remove element from front - O(1)
  • Front/Peek: View front element - O(1)
  • isEmpty: Check if queue is empty - O(1)
  • Size: Get number of elements - O(1)

Queue Implementation

Queue Using Python Deque (Best Approach)
from collections import deque

class Queue:
    """
    Queue implementation using deque.
    
    Deque (double-ended queue) gives O(1) operations
    on both ends, perfect for queue!
    
    Front of queue is at left (index 0)
    Rear of queue is at right (end)
    """
    
    def __init__(self):
        self.items = deque()
    
    def enqueue(self, item):
        """Add item to rear. Time: O(1)"""
        self.items.append(item)
        print(f"Enqueued: {item}")
    
    def dequeue(self):
        """Remove and return front item. Time: O(1)"""
        if self.is_empty():
            raise IndexError("Dequeue from empty queue")
        item = self.items.popleft()
        print(f"Dequeued: {item}")
        return item
    
    def front(self):
        """View front item without removing. Time: O(1)"""
        if self.is_empty():
            raise IndexError("Peek from empty queue")
        return self.items[0]
    
    def is_empty(self):
        """Check if queue is empty. Time: O(1)"""
        return len(self.items) == 0
    
    def size(self):
        """Get number of items. Time: O(1)"""
        return len(self.items)
    
    def display(self):
        """Display queue contents"""
        print(f"Queue: {list(self.items)}")

# Example usage:
queue = Queue()
queue.enqueue(1)
queue.enqueue(2)
queue.enqueue(3)
queue.display()  # Queue: [1, 2, 3]

print(f"Front: {queue.front()}")  # Front: 1
queue.dequeue()  # Dequeued: 1 (first in, first out!)
print(f"New front: {queue.front()}")  # New front: 2

Queue Variations

1. Circular Queue

A circular queue connects the end back to the beginning, making efficient use of space.

Circular Queue Implementation
class CircularQueue:
    """
    Fixed-size circular queue using array.
    When rear reaches end, it wraps around to beginning.
    """
    
    def __init__(self, capacity):
        self.capacity = capacity
        self.queue = [None] * capacity
        self.front = 0
        self.rear = -1
        self.count = 0
    
    def enqueue(self, item):
        if self.is_full():
            print("Queue is full!")
            return
        
        self.rear = (self.rear + 1) % self.capacity
        self.queue[self.rear] = item
        self.count += 1
    
    def dequeue(self):
        if self.is_empty():
            raise IndexError("Queue is empty")
        
        item = self.queue[self.front]
        self.front = (self.front + 1) % self.capacity
        self.count -= 1
        return item
    
    def is_empty(self):
        return self.count == 0
    
    def is_full(self):
        return self.count == self.capacity

2. Priority Queue

Elements are served based on priority, not order of arrival.

Priority Queue Using Heap
import heapq

class PriorityQueue:
    """
    Priority Queue - elements with higher priority are served first.
    Uses min-heap (lower values = higher priority)
    """
    
    def __init__(self):
        self.heap = []
    
    def enqueue(self, item, priority):
        """Add item with priority. Lower number = higher priority"""
        heapq.heappush(self.heap, (priority, item))
    
    def dequeue(self):
        """Remove and return highest priority item"""
        if self.is_empty():
            raise IndexError("Queue is empty")
        return heapq.heappop(self.heap)[1]
    
    def is_empty(self):
        return len(self.heap) == 0

# Example:
pq = PriorityQueue()
pq.enqueue("Low priority task", 5)
pq.enqueue("High priority task", 1)
pq.enqueue("Medium priority task", 3)

print(pq.dequeue())  # High priority task (priority 1)
print(pq.dequeue())  # Medium priority task (priority 3)

Queue Applications

Breadth-First Search (BFS)

One of the most important queue applications - exploring a graph level by level.

BFS Using Queue
from collections import deque

def bfs(graph, start):
    """
    Breadth-First Search traversal.
    Visits all nodes at distance k before visiting nodes at distance k+1.
    
    Time: O(V + E) where V = vertices, E = edges
    Space: O(V) for queue and visited set
    """
    visited = set()
    queue = deque([start])
    visited.add(start)
    result = []
    
    while queue:
        node = queue.popleft()
        result.append(node)
        
        # Visit all neighbors
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    
    return result

# Example graph (adjacency list)
graph = {
    1: [2, 3],
    2: [1, 4, 5],
    3: [1, 6],
    4: [2],
    5: [2],
    6: [3]
}

print("BFS traversal:", bfs(graph, 1))
# Output: [1, 2, 3, 4, 5, 6]

# Hash Tables - Fast Lookups with O(1)

What is a Hash Table?

A hash table (also called hash map or dictionary in Python) stores key-value pairs and provides O(1) average-case lookup, insert, and delete operations!

Real-world analogy: A library catalog system. Instead of searching through every book, you look up the book's location code (key) to find it instantly.

Why Hash Tables Are Powerful:
  • Lightning Fast: O(1) average lookup/insert/delete
  • Flexible Keys: Use any immutable type as key
  • Counts & Frequencies: Perfect for counting occurrences
  • Caching: Store computed results for quick retrieval
  • Interview Favorite: ~30% of problems use hash tables!
üìä Hash Table Internal Structure Visualization

How Hash Tables Work

Hash tables use a hash function to convert keys into array indices:

  1. Hash function takes a key and computes an index: index = hash(key) % array_size
  2. Store the value at that index in the array
  3. To retrieve: hash the key again to find the index
Simple Hash Table Concept
# How hash tables work internally (simplified)

def simple_hash(key, size):
    """Convert key to array index"""
    return hash(key) % size

# Example:

print(simple_hash("banana", array_size))  # Might give: 7

my_dict = {}

print(my_dict["apple"])  # O(1) lookup ‚Üí 5

Count Character Frequencies
def count_frequencies(text):
    """
    Count frequency of each character.
    
    Example: "hello" ‚Üí {'h': 1, 'e': 1, 'l': 2, 'o': 1}
    
    Time: O(n), Space: O(k) where k is unique characters
    """
    freq = {}
    for char in text:
        freq[char] = freq.get(char, 0) + 1
    return freq

Pattern 2: Two Sum with Hash Map

Two Sum - Hash Map Solution
def two_sum(nums, target):
    """
    Find two numbers that add up to target.
    
    Naive: O(n¬≤) with nested loops
    Hash Map: O(n) with single pass!
    
    Example: nums = [2, 7, 11, 15], target = 9
    Answer: [0, 1] (indices of 2 and 7)
    
    Time: O(n), Space: O(n)
    """
    seen = {}  # Store: number ‚Üí index
    
    for i, num in enumerate(nums):
        complement = target - num
        
        if complement in seen:
            return [seen[complement], i]
        
        seen[num] = i
    
    return []

# Test
nums = [2, 7, 11, 15]
target = 9
result = two_sum(nums, target)
print(f"Indices: {result}")  # [0, 1]
print(f"Numbers: {nums[result[0]]}, {nums[result[1]]}")  # 2, 7

Pattern 3: Group Anagrams

Group Anagrams Using Hash Map
def group_anagrams(words):
    """
    Group words that are anagrams of each other.
    
    Example: ["eat", "tea", "tan", "ate", "nat", "bat"]
    Output: [["eat", "tea", "ate"], ["tan", "nat"], ["bat"]]
    
    Key insight: Anagrams have same sorted characters!
    
    Time: O(n * k log k) where n = words, k = avg word length
    Space: O(n * k)
    """
    anagram_groups = {}
    
    for word in words:
        # Sort characters to create key
        key = "".join(sorted(word))
        
        if key not in anagram_groups:
            anagram_groups[key] = []
        
        anagram_groups[key].append(word)
    
    return list(anagram_groups.values())

# Test
words = ["eat", "tea", "tan", "ate", "nat", "bat"]
result = group_anagrams(words)
print(result)
# [['eat', 'tea', 'ate'], ['tan', 'nat'], ['bat']]

Pattern 4: Longest Substring Without Repeating Characters

Sliding Window + Hash Map
def length_longest_substring(s):
    """
    Find length of longest substring with all unique characters.
    
    Example: "abcabcbb" ‚Üí 3 ("abc")
    
    Uses sliding window + hash map to track last seen index
    
    Time: O(n), Space: O(min(n, m)) where m is charset size
    """
    last_seen = {}
    max_length = 0
    start = 0
    
    for end in range(len(s)):
        char = s[end]
        
        # If char was seen and is in current window
        if char in last_seen and last_seen[char] >= start:
            start = last_seen[char] + 1
        
        last_seen[char] = end
        max_length = max(max_length, end - start + 1)
    
    return max_length

# Test
print(length_longest_substring("abcabcbb"))  # 3
print(length_longest_substring("bbbbb"))     # 1
print(length_longest_substring("pwwkew"))    # 3

Hash Table Complexities

Search: O(1) avg Insert: O(1) avg Delete: O(1) avg Worst case: O(n)

*Worst case O(n) happens with many hash collisions (rare with good hash function)

üå≥ Trees - Hierarchical Data Structures

What is a Tree?

A tree is a hierarchical data structure consisting of nodes connected by edges. It's like an upside-down tree with the root at the top!

Real-world examples:

  • üóÇÔ∏è File system (folders within folders)
  • üë®‚Äçüë©‚Äçüëß‚Äçüë¶ Family tree (genealogy)
  • üè¢ Organization hierarchy
  • üåê HTML DOM (web page structure)
  • üéØ Decision trees in AI/ML

Tree Terminology

Key Terms:
  • Root: Top node (no parent)
  • Parent: Node with children below it
  • Child: Node below a parent
  • Leaf: Node with no children
  • Height: Longest path from root to leaf
  • Depth: Distance from root to node
  • Subtree: Tree formed by a node and its descendants

Tree Traversal Methods

üìä Tree Traversal Methods Visualization

There are multiple ways to visit all nodes in a tree:

1. Inorder Traversal (Left ‚Üí Root ‚Üí Right)

Inorder Traversal
def inorder_traversal(root):
    """
    Visit nodes in order: Left ‚Üí Root ‚Üí Right
    
    For BST, this gives sorted order!
    
    Example tree:
        2
       / \
      1   3
    
    Inorder: [1, 2, 3]
    
    Time: O(n), Space: O(h) for recursion stack
    """
    result = []
    
    def traverse(node):
        if not node:
            return
        
        traverse(node.left)    # Visit left subtree
        result.append(node.data)  # Visit root
        traverse(node.right)   # Visit right subtree
    
    traverse(root)
    return result

2. Preorder Traversal (Root ‚Üí Left ‚Üí Right)

Preorder Traversal
def preorder_traversal(root):
    """
    Visit nodes: Root ‚Üí Left ‚Üí Right
    
    Good for creating a copy of the tree
    
    Example tree:
        2
       / \
      1   3
    
    Preorder: [2, 1, 3]
    
    Time: O(n), Space: O(h)
    """
    result = []
    
    def traverse(node):
        if not node:
            return
        
        result.append(node.data)  # Visit root FIRST
        traverse(node.left)    # Then left
        traverse(node.right)   # Then right
    
    traverse(root)
    return result

3. Postorder Traversal (Left ‚Üí Right ‚Üí Root)

Postorder Traversal
def postorder_traversal(root):
    """
    Visit nodes: Left ‚Üí Right ‚Üí Root
    
    Good for deleting trees (delete children before parent)
    
    Example tree:
        2
       / \
      1   3
    
    Postorder: [1, 3, 2]
    
    Time: O(n), Space: O(h)
    """
    result = []
    
    def traverse(node):
        if not node:
            return
        
        traverse(node.left)    # Left first
        traverse(node.right)   # Then right
        result.append(node.data)  # Root LAST
    
    traverse(root)
    return result

4. Level Order Traversal (Breadth-First)

Level Order Traversal (BFS)
from collections import deque

def level_order_traversal(root):
    """
    Visit nodes level by level (left to right).
    
    Example tree:
        1
       / \
      2   3
     / \
    4   5
    
    Level order: [1, 2, 3, 4, 5]
    
    Time: O(n), Space: O(w) where w is max width
    """
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        node = queue.popleft()
        result.append(node.data)
        
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)
    
    return result

Common Tree Problems

Maximum Depth of Binary Tree

Find Maximum Depth (Height)
def max_depth(root):
    """
    Find the maximum depth (height) of tree.
    
    Example:
        3
       / \
      9  20
        /  \
       15   7
    
    Max depth: 3
    
    Time: O(n), Space: O(h)
    """
    if not root:
        return 0
    
    left_depth = max_depth(root.left)
    right_depth = max_depth(root.right)
    
    return max(left_depth, right_depth) + 1

Check if Tree is Balanced

Balanced Binary Tree Check
def is_balanced(root):
    """
    A tree is balanced if heights of left and right
    subtrees differ by at most 1 for ALL nodes.
    
    Time: O(n), Space: O(h)
    """
    def check_height(node):
        if not node:
            return 0
        
        left_height = check_height(node.left)
        if left_height == -1:
            return -1
        
        right_height = check_height(node.right)
        if right_height == -1:
            return -1
        
        if abs(left_height - right_height) > 1:
            return -1
        
        return max(left_height, right_height) + 1
    
    return check_height(root) != -1

Invert Binary Tree

Invert (Mirror) Binary Tree
def invert_tree(root):
    """
    Invert a binary tree (mirror it).
    
    Before:      After:
        4            4
       / \          / \
      2   7        7   2
     / \ / \      / \ / \
    1  3 6  9    9  6 3  1
    
    Time: O(n), Space: O(h)
    """
    if not root:
        return None
    
    # Swap left and right children
    root.left, root.right = root.right, root.left
    
    # Recursively invert subtrees
    invert_tree(root.left)
    invert_tree(root.right)
    
    return root
